=== DESIGN PRINCIPLES ===

1. Sacred Calm — generous whitespace, slow/gentle animations, no visual noise.
2. Meaning before price — always show pooja meaning, pujari, and temple before showing the price.
3. Devotee-first language — 'Offer This Pooja', 'Your Seva Has Been Offered', 'Seva Basket'.
4. Post-pooja reflection — after a pooja, create a moment of stillness before showing CTAs.
5. Never use ALL CAPS except for the brand name DEVASEVA in logos/headers.
6. Rounded corners throughout: 8px for small components, 12px for cards, 16px for large panels.
7. No harsh shadows — use 0px 1px 3px rgba(0,0,0,0.08) for light mode cards.
8. Minimum touch target: 44x44px for all interactive elements on mobile.
